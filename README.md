# 서울디지텍고등학교 AI 프로젝트


SmartSpace 동아리에서 개발한 AIY를 이용한 음성 비서입니다. 


## 1. 개발언어

>Python 3.6.3

## 2. 사용된 API

* 미세먼지 측정
* 버스 데이터 
* 날씨
* 급식 데이터 (파싱)

## 3. 제공하는 서비스

* 중식 / 석식 메뉴 조회
* 이번 달 학사 일정 조회
* 버스 도착 예정 시간 조회
* 오늘의 날씨 조회
* 초미세먼지 , 미세먼지 조회
